# 🚀 AI Email Assistant with Voice Command

AI Email Assistant is a smart, offline-capable email processing system built using Python and Streamlit.
It helps users automatically analyze, classify, and manage emails efficiently without relying on external APIs.

The system integrates Natural Language Processing (NLP) techniques with rule-based logic to provide features such as email classification, spam detection, summarization, priority scoring, and smart reply generation.

Additionally, it supports **offline voice commands** using VOSK, allowing users to interact with the system hands-free.

---

## 🔥 Key Features

* 📌 Email Classification (Important / Spam / Normal)
* 🧠 Priority Scoring (0–100)
* 📝 Email Summarization
* 💬 Smart Reply Suggestions
* 🎤 Voice Command Support (Offline)
* 🔐 Privacy First (No API usage)
* 📊 Interactive Dashboard
* 🎨 Modern Glass UI (Streamlit)

---

## 🛠️ Technologies Used

* **Python**
* **Streamlit** (Frontend Dashboard)
* **VOSK** (Offline Speech Recognition)
* **Pyttsx3** (Text-to-Speech)
* **Pandas** (Data Handling)
* **Custom Rule-Based NLP**

---

## 💡 How It Works

1. User inputs email text or uses voice command
2. Email is processed through multiple modules:

   * Spam Detection
   * Classification
   * Summarization
   * Priority Scoring
3. System generates:

   * Category (Spam / Important / Normal)
   * Summary
   * Priority Score
   * Suggested Reply
4. Voice module provides:

   * Speech input
   * Audio response output
